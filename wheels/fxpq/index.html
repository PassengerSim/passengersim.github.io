PassengerSim User Instructions

Install Conda (Miniforge)

If you do not already have conda installed on your computer, you can install it by running the code below. This should only be necessary once per computer (or per user account on the computer).

curl -L -O "https://github.com/conda-forge/miniforge/releases/latest/download/Miniforge3-$(uname)-$(uname -m).sh"
bash Miniforge3-$(uname)-$(uname -m).sh

Install the Environment

If you are installing on MacOS, run this command in the Terminal:

conda env update -n SKY -f https://www.passengersim.com/wheels/fxpq/macOS.yml

Or, if you are installing for Linux, run this command: 

conda env update -m SKY -f https://www.passengersim.com/wheels/fxpq/linux.yml

The command will install a conda environment called SKY that contains the passengersim package. To confirm that the environment was installed correctly, run the following command:

conda run -n SKY python -m passengersim info

You should see a report showing the version of the passengersim package that is installed, as well as its expiration date.

Running PassengerSim

To use PassengerSim, first change to the directory where you want to work, then run Jupyter lab from your terminal like this:

conda run -n SKY jupyter lab
