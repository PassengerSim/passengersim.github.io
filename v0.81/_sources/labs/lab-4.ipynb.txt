    "\n",
    "In this lab, we will look at implementing a new idea in PassengerSim. Or, we will pretend our\n",
    "idea is \"new\", but actually we'll borrow someone else's existing idea.\n",
    "We will loosely follow an idea previously explored by the PODS consortium: \n",
    "[\"Modeling the Effects of User Influence in Revenue Management\", Eric Hao, May 2013](https://sites.gatech.edu/ce-atlatgt/files/2023/01/201306-L-Modeling_Effects_of_User_Influence_in_RM.pdf)\n",
    "\n",
    "> The user compares current loads on a leg at a time frame to a historical sample (or set of samples).\n",
    "> - If the current load is too low, adjust the forecast to be lower, decreasing seat protection in order to accept more bookings.\n",
    "> - If the current load is too high, adjust forecast to be higher, increasing seat protection to value high fare passengers.\n",
    "\n",
    "In previous PODS work, this idea was called \"user influence\", or \"dynamic user influence\" (DUI).  \n",
    "For this work, we want to rename the idea, partly because \"DUI\" means something else\n",
    "to many people, but moreso to remove some ambiguity, as the \"user\" here doesn't actually refer to the\n",
    "PassengerSim user, but rather to a theoretical RM analyst who would be using some tool\n",
    "to tinker with the RM system while it is running.  So we will call this idea a\n",
    "\"heuristic adjustment\".\n",
    "\n",
    "Note this lab is *not* meant as any kind of proof for whether this idea is good or bad, \n",
    "but rather as a technical guide for how to work with PassengerSim to implement and test an idea.\n",
    "We selected this idea to explore because it will exercise two different sides of using \n",
    "PassengerSim: tinkering with outputs to create a bespoke analysis of simulation results,\n",
    "and tinkering with the inputs and algorithms to change how the simulation runs.\n",
    "\n",
    "To get started, we'll load the config for the standard 3MKT demo model to work with."

    "The idea of what we want to do has two parts:\n",
    "\n",
    "- Get some idea of what the range of \"normal\" booked load factor looks like over the booking curve. \n",
    "- While running a simulation, compare the current booked load factor at any point to the \"normal\" range, and apply an intervention if we are running too hot or too cold.\n",
    "\n",
    "You could think about doing all of this endogenously within the simulation, but  \n",
    "for this lab, we will do the first step by making a static analysis of simulation results,\n",
    "and then provide those curves back to the simultion as an exogenous set of \"normal\" curves.\n",
    "In some respects this is \"cheating\" by using too much knowledge, but since this lab is \n",
    "about learning the mechanics of PassengerSim, we'll allow it.\n",

    "def collect_blf_detail(sim: pax.Simulation, days_prior: int) -> dict | None:\n",
    "    \"\"\"A daily callback function to collect booked load factor for all legs.\"\"\"\n",
    "    if sim.eng.sample < sim.eng.burn_samples:\n",
    "        # don't collect data during burn period\n",
    "        return None\n",
    "    # create a dictionary to store the booked load factor for each leg\n",
    "    out = {}\n",
    "    # iterate over all legs in the simulation engine,\n",
    "    # and collect the current booked load factor\n",
    "    for leg in sim.eng.legs:\n",
    "        key = leg.leg_id\n",
    "        out[f\"leg-{key}\"] = leg.sold / leg.capacity\n",
    "    # return the value, which will be end up as \"callback_data\" in the results\n",
    "    return out"
   ]

    "def nest_dict(df: pd.DataFrame) -> dict:\n",
    "    assert isinstance(df.index, pd.MultiIndex), \"DataFrame index must be a MultiIndex\"\n",
    "    flat_dict = df.to_dict(orient=\"index\")\n",
    "    result = {}\n",
    "    for key, value in flat_dict.items():\n",
    "        target = result\n",
    "        # Traverse all keys but the last one\n",
    "        for k in key[:-1]:\n",
    "            target = target.setdefault(k, {})\n",
    "        # Assign the final value\n",
    "        target[key[-1]] = value\n",
    "    return result\n",
    "\n",
    "\n",
    "bounds_dict = nest_dict(bounds.set_index([\"leg_tag\", \"days_prior\"]))\n",
    "bounds_dict"

    "from passengersim.rm import RmAction\n",
    "\n",
    "SECONDS_PER_DAY = 24 * 60 * 60\n",
    "\n",
    "\n",
    "class BookedLoadFactor(RmAction):\n",
    "    frequency = \"daily\"\n",
    "\n",
    "    def __init__(\n",
    "        self,\n",
    "        *,\n",
    "        carrier: str,\n",
    "        minimum_sample: int = 10,\n",
    "        cfg: pax.Config | None = None,\n",
    "        adjustment_multiplier: float = 1.1,\n",
    "        min_days_between_actions: int = 7,\n",
    "    ):\n",
    "        \"\"\"\n",
    "        Apply an adjustment to leg forecasted demand when the bookings are atypical.\n",
    "\n",
    "        Parameters\n",
    "        ----------\n",
    "        carrier : str\n",
    "            Apply the adjustment only to legs for this carrier.\n",
    "        minimum_sample : int\n",
    "            Do not apply any adjustment until at least this many samples have\n",
    "            been simulated.\n",
    "        cfg : pax.Config | None\n",
    "            The configuration object for the simulation.  The booked load factor\n",
    "            curves will be stored in the `other_controls` section of this config.\n",
    "        adjustment_multiplier : float, default 1.1\n",
    "            The multiplier to apply to demand when bookings exceed the upper bound\n",
    "            curve of the booked load factor range. The inverse of this multiplier\n",
    "            will be applied when bookings fail to meet the lower bound curve.\n",
    "        \"\"\"\n",
    "        super().__init__(carrier=carrier, minimum_sample=minimum_sample, cfg=cfg)\n",
    "\n",
    "        self.hot_multiplier = adjustment_multiplier\n",
    "        \"\"\"The multiplier to apply when bookings exceed the upper bound.\"\"\"\n",
    "\n",
    "        self.cold_multiplier = 1.0 / adjustment_multiplier\n",
    "        \"\"\"The multiplier to apply when bookings fail to meet the lower bound.\"\"\"\n",
    "\n",
    "        self.latest_update_times = {}\n",
    "        \"\"\"Track the last time that an adjustment was applied to each leg.\"\"\"\n",
    "\n",
    "        self.min_days_between_actions = min_days_between_actions\n",
    "        \"\"\"Minimum wait period before applying another demand adjustment.\"\"\"\n",
    "\n",
    "    def reset(self, sim: pax.Simulation):\n",
    "        \"\"\"Clear all forecast multipliers for this carrier.\n",
    "\n",
    "        This is typically done at the end of a simulation sample (i.e. at\n",
    "        departure).\n",
    "        \"\"\"\n",
    "        self.latest_update_times.clear()\n",
    "        for leg in sim.eng.legs.set_filters(carrier=self.carrier):\n",
    "            for bucket in leg.buckets:\n",
    "                del bucket.forecast_multiplier\n",
    "\n",
    "    def run(self, sim: pax.Simulation, days_prior: int):\n",
    "        \"\"\"Run this action for the simulation and given days prior to departure.\"\"\"\n",
    "        if sim.eng.sample < self.minimum_sample:\n",
    "            return\n",
    "\n",
    "        # At the end of each sample, set all multipliers back to zero.\n",
    "        if days_prior == 0:\n",
    "            self.reset(sim)\n",
    "            return\n",
    "\n",
    "        # any leg that has been adjusted more recently than a cutoff time will be skipped\n",
    "        current_time = sim.eng.last_event_time\n",
    "        cutoff_time = current_time - self.min_days_between_actions * SECONDS_PER_DAY\n",
    "\n",
    "        for leg in sim.eng.legs.set_filters(carrier=self.carrier):\n",
    "            # for each leg, find the name of the appropriate BLF curve\n",
    "            leg_blf_curve = leg.tags.get(\"BLF_Curve\")\n",
    "            # if there is no BLF curve tag, apply no adjustment\n",
    "            if leg_blf_curve is None:\n",
    "                continue\n",
    "            # if this leg has been adjusted more recently than the cutoff time, skip it\n",
    "            if self.latest_update_times.get(leg.leg_id, 0) > cutoff_time:\n",
    "                continue\n",
    "            # find the upper and lower bounds for this leg and days_prior\n",
    "            ub = bounds_dict[leg_blf_curve][days_prior][\"upper_bound\"]\n",
    "            lb = bounds_dict[leg_blf_curve][days_prior][\"lower_bound\"]\n",
    "            # calculate the current booked load factor for this leg\n",
    "            leg_booked = leg.sold / leg.capacity\n",
    "            # if the leg is too hot or too cold, apply the appropriate adjustment\n",
    "            # and record the time of this adjustment in latest_update_times\n",
    "            if leg_booked > ub:\n",
    "                for bucket in leg.buckets:\n",
    "                    bucket.forecast_multiplier *= self.hot_multiplier\n",
    "                self.latest_update_times[leg.leg_id] = current_time\n",
    "            elif leg_booked < lb:\n",
    "                for bucket in leg.buckets:\n",
    "                    bucket.forecast_multiplier *= self.cold_multiplier\n",
    "                self.latest_update_times[leg.leg_id] = current_time"
   ]
