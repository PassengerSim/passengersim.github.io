     (function () {
       var bg = document.getElementById('parallax-bg');
       // Speed factor: 0 = completely fixed, 1 = scrolls at full speed (no parallax).
       // 0.35 makes the background drift upward at 35% of the page scroll rate,
       // producing a noticeable depth effect without losing the image too quickly.
       var speedFactor = 0.50;

       function onScroll() {
         // Prefer the MkDocs material scrolling container if present, fall back to window.
         var scrollY = window.scrollY || window.pageYOffset;
         bg.style.transform = 'translateY(' + (-scrollY * speedFactor) + 'px)';
       }

       // Attach to both window scroll and the MkDocs inner scroll container,
       // whichever is actually driving the page scroll in this theme.
       window.addEventListener('scroll', onScroll, { passive: true });
       var mdContent = document.querySelector('.md-content');
       if (mdContent) {
         mdContent.addEventListener('scroll', onScroll, { passive: true });
       }
     })();
   </script>

About Us
--------

PassengerSim is a revenue management simulation for passenger travel, developed by PassengerSim LLC with the support
of the `Air Transportation Laboratory at Georgia Tech (ATL\@GT) <https://sites.gatech.edu/ce-atlatgt/>`_
and its corporate members. Our mission is to provide a modern, fast, and customizable simulation environment for
researchers and practitioners to study ideas in revenue management. We are committed to advancing the state of the art
in this field through rigorous research and development.
